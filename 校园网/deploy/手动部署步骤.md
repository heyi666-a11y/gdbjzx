# 腾讯云 COS 手动部署步骤

## 方式一：使用 COSBrowser 客户端（最简单）

### 1. 下载 COSBrowser
- 访问：https://cosbrowser.cloud.tencent.com/
- 下载 Windows 版本并安装

### 2. 登录
- 使用腾讯云账号登录
- 或输入 SecretId 和 SecretKey

### 3. 创建存储桶
- 点击"创建存储桶"
- 名称：`campus-network-gdbjzx`
- 地域：广州（或其他）
- 访问权限：公有读私有写
- 点击创建

### 4. 上传文件
- 进入存储桶
- 点击"上传" → "选择文件夹"
- 选择：`d:\biancheng\校园网`
- 等待上传完成

### 5. 配置静态网站
- 点击存储桶名称进入详情
- 左侧菜单：基础配置 → 静态网站
- 开启静态网站
- 索引文档：`index.html`
- 错误文档：`index.html`
- 保存

### 6. 访问网站
- 静态网站 URL：`http://campus-network-gdbjzx.cos-website.ap-guangzhou.myqcloud.com`

## 方式二：使用腾讯云控制台

### 1. 登录控制台
- 访问：https://console.cloud.tencent.com/cos

### 2. 创建存储桶
- 点击"创建存储桶"
- 填写信息：
  - 名称：`campus-network-gdbjzx`
  - 地域：广州
  - 访问权限：公有读私有写
- 点击创建

### 3. 上传文件
- 进入存储桶 → 文件列表
- 点击"上传文件"或"上传文件夹"
- 选择网站文件夹上传

### 4. 配置静态网站（同上）

## 方式三：使用命令行（需要 tccli）

### 1. 安装 tccli
```bash
pip install tccli
```

### 2. 配置凭证
```bash
tccli configure
# 输入 SecretId、SecretKey、Region
```

### 3. 创建存储桶
```bash
tccli cos CreateBucket --Bucket campus-network-gdbjzx --Region ap-guangzhou --ACL public-read
```

### 4. 上传文件
```bash
# 安装 coscmd
pip install coscmd

# 配置
coscmd config -a 你的SecretId -s 你的SecretKey -b campus-network-gdbjzx -r ap-guangzhou

# 上传
coscmd upload -r d:/biancheng/校园网 /
```

### 5. 配置静态网站
```bash
tccli cos PutBucketWebsite --Bucket campus-network-gdbjzx --Region ap-guangzhou --WebsiteConfiguration '{"IndexDocument":{"Suffix":"index.html"},"ErrorDocument":{"Key":"index.html"}}'
```

## 可选：配置 CDN 加速

1. 进入腾讯云 CDN 控制台
2. 点击"添加域名"
3. 域名类型：静态加速
4. 源站类型：COS
5. 选择你的存储桶
6. 配置 HTTPS（免费申请 SSL 证书）
