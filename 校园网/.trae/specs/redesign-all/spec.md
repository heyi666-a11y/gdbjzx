# 广东北江中学校园网 - 设计与功能重做规格

## Why

用户反馈当前实现存在以下问题：
1. 存在虚假模拟数据，需要删除
2. 除了登录页和管理员页面外，其他页面设计不符合"白色简洁风"和"灵动"要求
3. 所有页面需要统一的设计语言和交互体验

## What Changes

### 必须保留的页面
- `index.html` - 登录页（已符合要求）
- `admin/dashboard.html` - 管理员页面（已符合要求）

### 必须重做的页面
1. `pages/home.html` - 首页
2. `pages/tieba.html` - 论坛
3. `pages/tieba-detail.html` - 帖子详情
4. `pages/channels.html` - 频道
5. `pages/channel-posts.html` - 频道帖子
6. `pages/messages.html` - 私信
7. `pages/chat.html` - 私聊
8. `pages/resources.html` - 资源中心
9. `pages/profile.html` - 个人中心

### 设计要求

#### 白色简洁风格
- 背景：纯白色 #ffffff
- 卡片：纯白色背景，轻微阴影
- 文字：深灰色 #1a1a1a（标题），#666666（正文）
- 分割线：极淡灰色 #f0f0f0

#### 灵动交互
- 按钮点击：轻微缩放 + 颜色变化，150ms ease-out
- 卡片悬停：轻微上浮 + 阴影加深
- 页面切换：淡入淡出效果
- 列表加载：逐条淡入动画
- Toast提示：从顶部滑入，3秒后滑出

#### 蓝紫渐变（仅用于关键强调）
- 用于：顶部导航背景、主要按钮渐变、VIP标识
- 渐变：#667eea → #764ba2
- 使用克制：仅在必要时使用，不可滥用

### 功能要求

#### 首页 (home.html)
- 用户欢迎信息 + 头像
- VIP状态标识
- 4个快捷入口卡片（论坛、频道、资源、消息）
- 最新帖子列表（从数据库获取，无模拟数据）
- 底部导航栏

#### 论坛 (tieba.html)
- 频道筛选（全部/学生/老师/综合）
- 帖子列表（从数据库获取）
- 发帖功能（弹窗表单）
- 点赞功能（实时更新）
- 点击进入帖子详情
- 无模拟数据

#### 帖子详情 (tieba-detail.html)
- 帖子完整内容
- 评论列表
- 评论输入
- 点赞功能
- 返回按钮

#### 频道 (channels.html)
- 三个频道卡片
- 权限控制显示
- 点击进入频道帖子

#### 私信 (messages.html)
- 会话列表
- 未读标识
- 发起私信功能
- 从数据库获取，无模拟数据

#### 私聊 (chat.html)
- 消息列表（自己/对方区分）
- 文字发送
- VIP图片发送
- 从数据库获取，无模拟数据

#### 资源中心 (resources.html)
- 资源列表
- 分类筛选
- 上传/下载（权限控制）
- 从数据库获取，无模拟数据

#### 个人中心 (profile.html)
- 用户信息
- VIP状态
- 统计数据
- 退出登录

### 数据要求
- **禁止使用任何模拟数据**
- 所有数据必须从 Supabase 实时获取
- 加载状态显示加载动画
- 网络错误显示友好提示
- 空状态显示引导内容

## Impact
- 所有页面需要重新设计样式
- 需要移除所有 mockData 相关代码
- 需要确保所有功能连接真实数据库

## Technical Requirements
- Supabase 实时数据获取
- 统一的错误处理
- 统一的加载状态
- 统一的空状态处理
- showToast 提示函数
- 底部导航栏
